# TOR-Routed-Shell
A small cross-platform Python utility that launches a new shell (terminal window) with all proxy environment variables preconfigured to route traffic through a local Tor SOCKS5 proxy.
---
**Requirements**
-Python 3.7+
-A running Tor instance using a local SOCKS5 proxy
-Default Tor Browser → Port 9150
-Tor system service → May use port 9050 (adjust in script if needed)
